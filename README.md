# Housing Market Analysis

This project analyzes housing price trends using Python and data science tools.

Tools used:
- Python
- Pandas
- Numpy
- Matplotlib
- Seaborn
- Sklearn

Key insights:
- Identified factors influencing price changes
- Built a regression model predicting housing prices
